# Build output

The signed **universal** release APK for Professor VPN is produced by the
GitHub Actions workflow `.github/workflows/build.yml` on every push to
`main` / `genspark_ai_developer`, and on manual dispatch.

- Output name: `ProfessorVPN-v<versionName>-universal.apk` (current: `ProfessorVPN-v6.1-universal.apk`)
- Download the **v6.1** APK from the
  [Releases page](https://github.com/aptixzero/PRF_VPN/releases/tag/v6.1)
  (single latest release, refreshed on every push to `main` / manual dispatch),
  or from the workflow run's **Artifacts**.

Only the latest version is referenced here; the previous
`ProfessorVPN-v6-universal.apk` has been removed in favour of v6.1.
